🗺️ Kenz Quest • كنز المغرب
An immersive cultural adventure through Morocco
🌟 Overview

Kenz Quest is an interactive cultural exploration platform designed to transform tourism into a gamified, educational, and immersive experience.

Users travel through real locations in Morocco (starting with Marrakech), solve riddles, complete quizzes, interact with WebAR experiences, and unlock real-world artisan activities.

💡 The goal: reconnect people—especially youth and tourists—with Morocco’s living heritage, crafts, and history.

🚀 Features
    🎮 Gamified Exploration
    Multi-step journey across cultural landmarks
    Riddles to unlock locations
    Score system and progression tracking
🧠 Knowledge Challenges
    General knowledge quizzes about Morocco 🇲🇦
    On-site verification questions (prove physical presence)
    Unlock next stages by completing challenges
🧞 WebAR Integration
    Augmented reality experiences using MyWebAR
    Interactive storytelling (Genie guide)
🏺 Real-World Experiences
    Jewelry crafting workshop 💍
    Zellige tile creation 🧩
    Moroccan cooking class 🍲
    Rug weaving 🧶
🔐 Partner System
    Access codes provided by real artisans/instructors
    Bridges digital experience with physical economy
🏆 Certificate System
    Personalized downloadable certificate at completion
🧱 Tech Stack
    Frontend / App Framework: Streamlit
    Language: Python
    UI/Styling: Custom CSS
    Media Handling: PIL (Pillow)
    AR Integration: MyWebAR (iframe embedding)
    Interaction: streamlit-image-coordinates

🎯 How It Works
    User selects a region on the map 🗺️
    Starts the journey with a Genie (WebAR) 🧞
    Solves a riddle to unlock the first location
    Explores story + cultural content
    Completes 4 quizzes (2 general + 2 on-site)
    Unlocks next stop or real-world experience
    Continues journey across multiple stops
    Finishes with a certificate of exploration 🏆
